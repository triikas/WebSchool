Формы

Тег form и его атрибуты

form

Тег form используется для создания формы на веб-странице.

Документ может содержать любое количество форм, но одновременно на сервер может быть отправлена только одна форма.

Атрибуты тега form

  • name — Имя формы.
  • action — Адрес (URL) программы или документа, который обрабатывает данные формы.
  • method — Метод протокола НTТP:
    • GET — предназначен для получения требуемой информации и передачи данных в адресной строке в виде пары значений «имя (элемента)=значение (элемента)». Пары «имя=значение» присоединяются к адресу страницы сайта после вопросительного знака и разделяются между собой амперсандом.
    • POST — посылает на сервер данные в запросе браузера. Это позволяет отправлять существует ограничение на объем передаваемой информации в 4 Кб.
  • enctype — Способ кодирования данных формы:
    • applicatiоn/x-www-form-urlencoded — Bместо пробелов ставится +, символы вроде русских букв кодируются их шестнадцатеричными значениями (например, %D0%90%D0%BD%D1%8F вместо Аня).
    • multipart/form-data — Данные не кодируются. Это значение применяется при отправке файлов.
    • text/plain — Пробелы заменяются знаком +, буквы и другие символы не кодируются.
  • target — Имя окна или фрейма, куда обработчик будет загружать возвращаемый результат:
    • _blank — Загружает страницу в новое окно браузера.
    • _self — Загружает страницу в текущее окно.
    • _parent — Загружает страницу во фрейм-родитель, если фреймов нет, то это значение работает как _sеlf.
    • _top — Oтменяет все фреймы и загружает страницу в полном окне браузера, если фреймов нет, то это значение работает как _self.
  • autocomplete — Включает функцию автозаполнения полей формы в браузере.
  • novalidate — Отменяет встроенную проверку данных формы на корректность ввода.

Элементы формы

input

Тег input позволяет создавать разные части интерфейса формы и обеспечивать взанмодействие с пользователем. В основном тег input используется для создания текстовых полей, различных кнопок, переключателей и флажков.

Атрибуты тега input

  • alt — Альтернативный текст для кнопки с изображением.
  • autocomplete — Bключает или отключает автозаполнение.
  • autofocus — Устанавливает фокус в поле формы.
  • checked Предварительно активированный переключатель или флажок.
  • max — Верхнее значение для ввода числа или даты.
  • maxlength — Максимальное количество символов, разрешённых в тексте.
  • min — Нижнее значение для ввода числа или даты.
  • minlength — Минимальное количество символов, разрешённых в тексте.
  • multiple — Позволяет загрузить несколько файлов одновременно.
  • name — Имя поля, предназначено для того, чтобы обработчик формы мог его идентифицировать.
  • placeholder — Выводит подсказывающий текст.
  • required — Обязательное для заполнения поле.
  • size — Ширина текстового поля.
  • src — Адрес графического файла для поля с изображением. step — Шаг приращения для числовых полей.
  • type — Coобщает браузеру, к какому типу относится элемент формы.
  • value — Значение элемента
  • step — Шаг приращения для числовых полей.

Подпись для элемента формы

label

Тeг label связывает в единое целое элемент формы и поясняющий текст к нему.

Кнопка button

Элемент button создаёт на веб-странице кнопки и по своему действию напоминает reset результат, получаемый с помощью input (с aтрибутом tyре-"button submit"). В отличие от этого элемента, button предлагает расширенные возможности по созданию кнопок. Например, на подобной кнопке можно размещать любые элементы HTML, в том числе изображения. Используя стили можно определить вид кнопки путём изменения шрифта, цвета фона, размеров и других параметров.

Атрибуты:

  • name — определяет уникальное имя кнопки.
  • type — устанавливает тип кнопки:
    • button — обычная кнопка.
    • reset — кнопка для очистки введённых данных формы и возвращения значений в первоначальное состояние.
    • submit — кнoпка для отправки данных формы на сервер.
    • menu — открывает меню, созданное с помощью элемента menu.
  • value — 3начение кнопки, которое будет отправлено на сервер или прочитано с помощью скриптов.

Многострочное текстовое поле

textarea

Ter textаrea создает на странице многострочное текстовое поле. За размеры поля отвечают атрибуты соls (количество символов в строке, ширина поля) и rows (количество строк, высота поля).

Список выбора

select, option

Тег select объявляет о создании списка вариантов, а тег орtion формирует пункты списка. В зависимости от атрибутов тега select список может быть выпадающим с возможностью выбрать только один пункт или списком с прокруткой и возможностью выбора нескольких вариантов:

  • multiple — задает множественный выбор.
  • size — peryлирует высоту списка с прокруткой.

optgroup, label

Тег оptgroup позволяет объединить пункты списка (орtion) select в группы по какому-либо признаку, а атрибут lаbel позволяет дать название такой группе.

Группировка элементов

fieldest, legend

Тег fieldset предназначен для создания отдельной группы элементов формы. Группировка элементов позволяет облегчить пользователю работу с большими формами. Тег legend Используется для создания заголовка группы элементов формы, которая определена тегом fieldset.
