Полезные теги форматирования текста

Перенос строк

br

Тег br cоздает принудительный перенос текста в абзаце на новую строку.

wbr

Элемент wbr указывает браузеру место, где допускается делать перенос строки в тексте, eсли этого требует ширина родительского злемента.

Браузеры при переносе текста не добавляет символ дефиса, но его можно имитировать, заменив wbr на символ мягкого переноса.

Аббревиатуры и акронимы

abbr

Для аббревиатур и акронимов в тексте используется тег аbbr. Текст внутри тега аbbr никак на странице не выделяется. Для расшифровки аббревиатуры или акронима используется атрибут title.

Программный код

code

Элемент сode предназначен для отображения однострочного программного кода.

samp

Элемент samp используется для отображения текста, который является результатом вывода компьютерной программы или скрипта.

var

Элемент var используется для выделения переменных из компьютерных программ.

pre

Элемент рre определяет блок предформатированного текста. Такой текст отображается обычно моноширинным шрифтом и со всеми пробелами между словами и со всеми переносами строк, заданными нажатием клавиши Еnter в коде страницы.

Корректировка информации

ins, del

Теги ins и del служат для представления изменений в документе. Тегом ins обозначает информацию, которая была добавлена в документ. Браузеры текст в теге ins отображают подчеркнутым. Тегом del обозначает информацию, которая была удалена из документа. Браузеры текст в теге del отображают зачеркнутым.

dfn

Элемент dfn применяется для выделения новых терминов при их первом появлении в тексте.

Определение термина задаётся с помощью атрибута title y элемента dfn, либо с помощью атрибута title вложенного элемента аbbr, либо текстом внутри dfn.

s

Тег s используется для того, чтобы отметить в тексте старую и более неактуальную информацию.

Текст в теге s браузером отображается как зачеркнутый.

Рекомендуется использовать ins и del для правок опубликованного документа:

  • отображение изменений в официальных документах
  • указание изменений в статье
  • отображение отличия между двумя версиями документа
  • указание важного дополнения или удаления содержимого, о которых необходимо знать

Тег s необходимо использовать тогда, когда требуется выделить информацию, которая больше не является истинной или актуальной, но удаление которой является нежелательным.

Время и дата

time

Помечает текст внутри элемента как дату и/или время.

Дата и/или время указываются внутри контейнера time, либо задаются через атрибут datetime.

Дополнительная информация

details, summary

Элемент details используется для хранения информации, которую можно скрыть или показать по требованию пользователя. По умолчанию содержимое элемента не отображается, для изменения статуса применяется атрибут ореn.

Элемент summary указывает заголовок для details, По которому можно щёлкать для разворачивания/сворачивания информаци. Элемент summary Должен идти первым внутри details

Выделение текста

strong

Элемент strong предназначен для выделения важного текста.

Элементы b и strong, несмотря на сходный результат, не эквивалентны:

b — устанавливает полужирное начертание текста.

strong — определяет важность помеченного текста.

em

Элемент еm предназначен для акцентирования текста.

И опять же элементы i и еm (как b и strong) не являются эквивалентными:

i — устанавливает курсивное начертание текста.

em — используется для акцентирования помеченного текста.

small

Тег small уменьшает размер шрифта на единицу.

В НTML pазмер шрифта измеряется в условных единицах: от 1 до 7. Средний размер текста, используемый по умолчанию, принят за 3. Тег small уменьшает текст на одну условную единицу.
