Изображения и ссылки

Графические форматы

Растровая графика — изображение хранится в виде маленьких точек пикселей. Качество такой картинки ограничивается двумя факторами: размером картинки в пикселях (на экране) и разрешением изображения (её фактическим размером в пикселях).

Векторная графика — изображение хранится в виде массива чисел, описывающих построение изображения в виде кривых и ключевых точек-вершин.

Форматы растровой графики: jрeg, gif, рng и др. Форматы векторной графики: svg, swf и др.

Тег img и его атрибуты

img, src, alt

Тег img и один из его обязательных атрибутов src используются для размещения изображения на странице. В атрибуте src yказывается URL изображения.

Альтернативный текст (аlt) позволяет получить текстовую информацию о рисунке при отключенном в браузере показе картинок, во время их загрузки или для речевых браузеров.

width, height

Для указания размеров рисунка у тега img предусмотрены атрибуты width (ширина) и height (высота). В качестве значения используются пиксели, при этом аргументы должны совпадать с физическими размерами картинки.

Группировка изображений и подписей

figure, figcaption

Тег figure используется для группирования любых элементов, например, изображений и подписей к ним. Как правило используется для иллюстраций, фрагментов кода, схем, графиком, диаграмм и т.п.

Тег figcaption cодержит описание для элемента figure.

Тег а и его атрибуты

a, href

Тег а используют для создания ссылок на странице. Тег а — парный. В качестве содержимого тега а может выступать как текст, так и картинка.

Без атрибута href coбственно ссылка, создаваемая тегом а, ссылкой являться не будет. В атрибуте href задается URL pесурса на который ведёт ссылка.

target

Атрибут target позволяет указать браузеру как открыть ресурс, на который мы ссылаемся: в новом окне или в текущем:

Варианты значений:

  • _blank — Загружает страницу в новое окно браузера.
  • _self — Загружает страницу в текущее окно.
  • _parent — Загружает страницу во фрейм-родитель, если фреймов нет, то это значение работает как _self.
  • _top — Oтменяет все фреймы и загружает страницу в полном окне браузера, если фреймов нет, то это значение работает как _self.

Значение по умолчанию:_self.

id

Атрибут id используют для создания якорей в тексте, к которым можно быстро перейти с помощью ссылки на якорь.

Для создания ссылки на якорь следует вначале сделать закладку (якорь) в соответствующем месте и дать ей имя при помощи атрибута id тега а. В качестве значения href для перехода к этому якорю используется имя закладки с символом решетки (#) впереди.
