Списки

Нумерованный (упорядоченный) список

ol, li

Тег оl формирует на html-странице список пронумерованных пунктов. Каждый пункт списка заключается в тег пункта списка — li.

Маркированный (неупорядоченный) список

ul, li

Тег ul используют для создания маркированного списка пунктов (точки или другие значки). Каждый пункт списка заключается в тег пункта списка — li.

Многоуровневый список

При создании многоуровневого списка используются простые нумерованные и/или маркированные списки. Многоуровневость достигается за счет вложения одного списка в текущий пункт другого списка.

При создании многоуровневого списка важно помнить, что непосредственно в тегах ol и ul могут располагаться только теги li.

Список определений

dl, dt, dd

Тег dl используют для создания списка определений для научных, технических и других специальных терминов.

Внутри тега dl непосредственно располагаются пары тегов dt и dd. В тег dt помещают термин, а в тег dd определение для этого термина.
