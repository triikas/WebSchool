Блочная модель и единицы измерения

Единицы

Для того, чтобы указать размер или отступ недостаточно числа нужны и единицы измерения, что логично. В CSS существует множество разных единиц, но большинство из них может пригодиться лишь в экстраординарных случаях, здесь же представлены лишь наиболее популярные.

  • % — Размер указывается в процентах от размера родительского элемента (того, внутри которого находится текущий элемент). Значения могут быть больше 100, а также десятичными дробями.
  • vh/vw — Размер указывается в процентах от размера области просмотра на экране. Соответственно первый — высота, второй — ширина.
  • еm — Размер шрифта от текущего. Проще говоря, шрифт без стилей имеет размер 1em. Если вы поставите ему размер 2em, он станет в 2 раза больше.
  • px — Размер указывается в пикселях. Поскольку пиксель — абсолютная единица, размер не будет зависеть от родительского элемента или размера экрана.

Если значение 0, единицы можно не писать.

Блочная модель

Все элементы HTML делятся на блочные и строчные.

Блочный — прямоугольная область на экране, которой можно задать ширину, высоту и отступы (по умолчанию блок занимает максимальную доступную ширину и минимальную высоту необходимую его наполнению). Несколько блоков располагаются в столбик. (Параграфы и заголовки — это блочные элементы)

Строчный — область, размер которой зависит только от размера наполнения. Таким элементам можно задать только горизонтальные отступы. Несколько строчных элементов располагаются в строчку, один за другим. Такими, например, являются ссылки и картинки.

Размеры

width

Устанавливает ширину элемента без учёта ширины границ и отступов. Значение auto автоматически установит ширину по наполнению.

height

Устанавливает высоту элемента без учёта ширины границ и отступов. Значение auto автоматически установит высоту по наполнению.

max/min

Как для ширины, так и для высоты можно указать два свойства: min-width и max-width. Если ширина элемента не указана жёстко (абсолютными единицами), то она будет меняться в зависимости от размера экрана. Min-width задаёт минимальную ширину элемента, т.е. пока ширина элемента больше минимальной — она зависит от размера экрана, как только она станет меньше, ширина станет равна минимальной, и на экране появится ползунок. Таким же образом для максимального значения и высоты.

Границы

border

Позволяет указывать в одном свойстве (в любом порядке) цвет, толщину и стиль границы. Также это всё можно указывать отдельно с помощью: border-color, border-style и border-width.

Стили:

К border можно через дефис написать направление, тогда граница будет только с указанной стороны. (top — верх, bottom — низ, right — право, left — лево)

border-radius

Отдельное свойство, создающее закруглённые углы у границы. В качестве значения указывается длина радиуса.

Отступы

padding

Устанавливает отступ от границы элемента внутрь. Если указать одно значение, оно будет применено ко всем 4 сторонам, а если указать 4 значения, то можно указать разные. Также, дописав через дефис направление, можно отдельным свойством указать отступ с одной стороны.

margin

Всё тоже самое, только отступы устанавливаются внешние.

Тень

box-shadow

Добавляет элементу тень.

Значения (через пробел):

  • none — Убирает тень.
  • insert — Тень выводится внутри элемента.
  • Сдвиг тени по горизонтали относительно элемента.
  • Сдвиг тени по вертикали относительно элемента.
  • Радиус размытия тени (по умолчанию 0).
  • Цвет.

Свойство display

display

Многоцелевое свойство, которое определяет, как элемент будет отображаться.

Основные значения:

  • block — Элемент отображается как блочный.
  • inline — Элемент отображается как строчный.
  • inline-block — Элемент отображается как блочный, но обтекаемый другими элементами как строчный.
  • none — Элемент не отображается. Страница формируется так, словно его нет.

Обтекание

float

Определяет по какой стороне будет выравниваться элемент, при этом остальные элементы будут обтекать его с оставшихся сторон. При значении none, элемент располагается как обычно. (варианты значений: left, right, none)

clear

Устанавливает с какой стороны элемент нельзя обтекать. Если указано float, то clear отменяет его действие для указанных сторон. (варианты значений: left, right, none, both(обе стороны))
