Изображения

picture

Контейнер для хранения нескольких элементов source, которые поддерживают элемент img. Этот элемент позволяет указывать разные изображения с учётом размера экрана, диоганалей, формата изображения и других параметров. Вот несколько областей применения picture:

  • выводить рисунки разного размера для мобильных и настольных устройств;
  • отображать изображения разных пропорций, учитывающих ориентацию устройства;
  • выводить изображение в векторном формате SVG, a для браузеров, его не поддерживающих, показывать РNG.

Внутри рicture cодержится ноль или несколько элементов source, которые идут перед одним элементом img.

Можно отображать изображения и через один только img
