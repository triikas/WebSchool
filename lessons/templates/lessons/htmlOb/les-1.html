Понятие тега и атрибута

Тег — команда, сообщающая браузеру о том, каким образом браузер должен отобразить данный фрагмент текста на экране. Название тега пишется в нижнем регистре.

Открывающий и закрывающий тег

Атрибуты тега

Атрибуты предоставляют дополнительную информацию о содержимом элемента. Название атрибута, как и название тега, пишется в нижнем регистре.

Большинство атрибутов используются только у определенных тегов. Есть лишь небольшое количество атрибутов, которые могут присутствовать в любом теге: сlаss, id, lang, title...

Структура кода страницы

  • !doctype html — сообщает браузеру о версии языка html с помощью которой сверстана данная страница.
  • html — в этом парном теге располагается код всей веб-страницы.
  • head — в этом парном теге располагается служебная информация для браузера, которая не отображается на странице.
  • body — в этом парном теге располагается все отображаемое в браузере содержимое страницы.
