Адресация в HTML

Основа основ интернета возможность быстро перемещаться от одного документа к другому. Мы кликаем на ссылку и вот перед нами уже совсем другая страница. Для размещения картинки или видеоролика на странице нужно указать ссылку на нужный медиа-файл, чтобы подключить таблицы стилей — ссылку на сss-файл.

Что бы все ссылки работали правильно нужно правильно указать адрес документа (URL). Существует два типа адресации:

  • Абсолютная.
  • Относительная.

Абсолютный адрес

Абсолютная адресация применяется в случае, когда нужно сослаться на файл, расположенный на другом сайте. Нужно указать полный адрес документа, включая протокол доступа (как правило, https://), название сайта (доменное имя) и путь к файлу на сайте.

Правильно

  • https://www.fft.ru/ОpenPage/point 1/
  • https://w3c.org

Неправильно

  • www.fft.ru/ОpenPage/point 1/ — не указан протокол доступа
  • https://ОpenPage/point 1/ — не указано название сайта

Для задания абсолютного адреса необходимо, чтобы файлы сайта были опубликованы в интернете.

Корневая папка сайта

Корневая папка сайта — это главная или основная папка. В ней находятся файлы, обеспечивающие работоспособность сайта. Корневая папка сайта может быть единственной, когда сайт имеет простую структуру, но чаще она содержит другие вложенные папки и файлы.

Корневая папка отличается от вложенных в нее папок тем, что именно в ней лежит html-файл главной страницы сайта.

Файлу главной страницы сайта нередко присваивают либо имя index, либо default (хотя это не принципиально).

На самом деле, никто вас не заставляет структурировать сайт именно таким образом. Это просто общепринятая версия.

Относительный адрес

Относительная адресация применяется в случае, когда нужно сослаться с одного файла сайта на другой файл этого же сайта. Этот тип адресации используется намного чаще.

Рассмотрим примеры построения относительных URL для корневой папки сайта следующего вида:

Примеры адресов:

1. Сделать ссылку со страницы index.html из корневой папки на страницу history.html history.html
2. Сделать ссылку со страницы index.html из корневой папки на страницу index.html в папке contacts contacts/index.html
3. Сделать ссылку со страницы index.html из корневой папки на страницу index.html в папке vacancy about/vacancy/index.html
4. Сделать ссылку со страницы index.html из корневой папки на страницу index.html в папке director about/structure/director/index.html
5. Сделать ссылку со страницы index.html папке about на страницу index.html корневой папки ../index.html
6. Сделать ссылку со страницы index.html в папке structure на страницу history.html ../../history.html
7. На страницy index.html в папке management вставить картинку logo.jpg ../../../img/logo.jpg
8. К странице іndex.html в папке vacancy прилинковать файл стилей рrint.css ../../css/print.css
