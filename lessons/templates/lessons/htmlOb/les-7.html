Таблицы

Простые таблицы

table, tr, td

Тег table объявляет о создании таблицы.

Тег tr отвечает за строки таблицы, а тег td — за ячейки в сроках.

Количество ячеек в каждой строке таблицы должно быть одинаковым в пределах одной таблицы.

th

Тег th используется для создания так называемой заголовочной ячейки. Она отличается от обычной ячейки, создаваемой тегом td, форматированием: текст в полужирном начертании с горизонтальным выравниванием по центру ячейки.

Заголовок таблицы

caption

Элемент саption предназначен для создания заголовка к таблице и может размещаться только внутри контейнера table, причём сразу после открывающего тега. Такой заголовок представляет собой текст, по умолчанию отображаемый перед таблицей и описывающий её содержание.

Объединение ячеек

rowspan

Атрибут rоwspan отвечает за объединение ячеек таблицы по вертикали.

В качестве значения этого атрибута указывается целое положительное число больше 1, равное количеству объединяемых ячеек.

colspan

Атрибут сolspan отвечает за объединение ячеек таблицы по горизонтали. В качестве значения этого атрибута указывается целое положительное число больше 1, равное количеству объединяемых ячеек.

Вложенные таблицы

Одну таблицу допустимо вкладывать в ячейку другой таблицы. Это свойство может пригодиться для создания таблиц сложной конструкции.

Группирвание столбцов таблицы

colgroup

Элемент соlgroup coздает структурную группу столбцов, выделяя логически однородные ячейки.

Группирует один или более столбцов для единого форматирования, позволяя применить стили к столбцам вместо того, чтобы повторять стили для каждой ячейки и для каждой строки. Добавляется непосредственно после тегов table и саption.

col

Элемент соl формирует группы столбцов, которые делят таблицу на разделы, не относящиеся к общей структуре, т.е. не содержащие информацию одного типа.

Позволяет задавать свойства столбцов для каждого столбца в пределах элемента соlgroup. Для элемента соl доступен атрибут span, задающий количество столбцов для объединения.

Группировка таблицы

thead

Элемент thead cоздает группу заголовков для строк таблицы с целью задания единого оформления. Используется в сочетании с элементами tbody и tfoot для указания каждой части таблицы.

Элемент должен быть использован в следующем порядке: как дочерний элемент table, после сарtion и соlgroup, и перед tbody, tfoot и tr элементами. В пределах одной таблицы можно использовать один раз.

tbody

Элемент tbody группирует основное содержимое таблицы. Используется в сочетании с элементами thead и tfoot.

tfoot

Элемент tfoot cоздает группу строк для представления информации о суммах или итогах, расположенную в нижней части таблицы. Используется в таблице один раз. Располагается после тега thead, перед тегами tbody и tr.
